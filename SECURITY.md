# Security and responsible disclosure

The Arithmon repositories hold research code, formal proofs, and documents. If
you find a security problem, a vulnerability in published code, or a sensitive
issue you would rather not file in public, please report it privately by email
to brieuc@bdelaf.com instead of opening a public issue.

For ordinary scientific corrections (a wrong number, a broken derivation, a
citation error), a public issue or pull request is welcome.
